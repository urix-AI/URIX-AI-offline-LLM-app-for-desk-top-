# URIX AI

URIX AI is an offline AI assistant and productivity platform designed to run Large Language Models (LLMs) locally on user hardware without requiring cloud-based AI services.

The project focuses on privacy, speed, and full local control while providing advanced AI capabilities such as content generation, document processing, presentation creation, and intelligent automation.

## Features

- Offline Local LLM Execution
- Privacy-First Architecture
- AI-Powered Content Generation
- PowerPoint Presentation Generation
- Document Processing
- Modern Desktop Interface
- Template-Based Workflows
- Windows Desktop Support
- Extensible Plugin-Friendly Design

## Technology Stack

- Python
- PyQt
- Ollama / Local LLM Backends
- Transformers
- PyTorch
- PowerPoint Automation Libraries

## Objectives

URIX AI aims to provide users with a powerful local AI environment that can perform advanced language and productivity tasks without sending sensitive data to external servers.
